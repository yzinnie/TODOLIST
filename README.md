<h2> 🦾 주요 기능 </h2>

 #### 로그인 기능
- 투두리스트를 작성하기 전에, 원하는 이름으로 로그인할 수 있어요.
- 사용자의 이름은 localStorage에 저장됩니다.

#### 오늘의 할 일 작성 & 삭제
- 오늘의 투-두리스트를 작성합니다.
- 마음에 들지 않거나, 이미 끝난 일은 삭제할 수 있어요.
  
#### 오늘 날씨
- 현재 위치를 허용한다면, 오늘의 날씨를 알려드려요.
- 지역명과 함께 표시됩니다.
- 겨울에는 화면에 눈이 내립니다. ⛄
